### JAVA

枚举类通过值获取对象

```java
public static TradeStatus getByValue(int value) {
    for (TradeStatus tradeStatus : values()) {
        if (tradeStatus.getValue() == value) {
            return tradeStatus;
        }
    }
    return null;
}
```
